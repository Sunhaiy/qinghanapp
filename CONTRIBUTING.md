# Contributing

感谢你愿意参与这个项目。

## 在开始之前

建议先确认这几件事：

1. 你准备解决的问题是什么
2. 这个问题是否已经有 Issue
3. 你的改动是否会影响线上真实接口联调

## 本地开发建议

1. 使用 Android Studio 最新稳定版
2. 使用 JDK 21
3. 确认 `local.properties` 中的后端地址配置正确
4. 优先在真实接口环境下验证，而不是只看本地 mock

## 提交代码前

请尽量保证：

- 能正常编译
- 关键交互至少手动回归一遍
- 不要把无关改动混进同一个提交
- UI 调整尽量附带截图

## Pull Request 建议

PR 描述里建议至少写清楚：

- 改了什么
- 为什么改
- 影响哪些页面或接口
- 如何验证
- 是否存在已知风险

## 沟通方式

如果你不确定某个交互或接口应该怎么接，优先开 Issue 或在 PR 里说明你的假设，不要默默绕过真实问题。
